# SEO Content Generation Guidelines

This file provides additional context for Claude when generating SEO content for Islander Orthodontics.

## Brand Voice & Tone
- Professional yet warm and approachable
- Focus on patient comfort and care
- Emphasize expertise and modern technology
- Use inclusive, welcoming language

## SEO Best Practices
- Target keyword density: 1-2% (natural integration)
- Include location-specific terms (Mercer Island, Seattle area)
- Use semantic keywords and LSI terms
- Header hierarchy: H1 (once) → H2 → H3
- Meta descriptions: 150-160 characters
- Title tags: Under 60 characters, include location

## Content Structure Preferences
- Hero sections should be concise (1-2 sentences max)
- Use bullet points for benefits/features
- Include clear CTAs (e.g., "Schedule a Consultation")
- FAQ sections when applicable
- Include trust signals (certifications, years of experience)

## Common Services & Terms
- Invisalign (capitalize correctly)
- Traditional braces
- Clear aligners
- Orthodontic treatment
- Smile transformation
- Board-certified orthodontist

## Location Context
- Primary location: Mercer Island, WA
- Service area: Greater Seattle area
- Nearby cities: Bellevue, Seattle, Issaquah

## Schema Markup
- Always include LocalBusiness schema
- Add FAQPage schema when FAQ content exists
- Include proper address, phone, and business hours in schema
